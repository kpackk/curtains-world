# Дизайн: Комплексное улучшение сайта Curtains World

**Дата:** 2026-02-24
**Цели:** SEO + органический трафик, повышение конверсий
**Аудитория:** русскоязычные экспаты в Дубае
**Объём изменений:** полная переработка (без ограничений)

---

## Секция 1: SEO-оптимизация

### 1.1 Мета-теги и технический SEO
- Canonical-теги на каждой странице
- Twitter Card мета-теги (отсутствуют)
- Проверка hreflang связки ru ↔ en
- Расширение robots.txt: ссылка на sitemap, запрет технических путей

### 1.2 Семантическая HTML-разметка
- Замена `<div>` заголовков на `<h1>`-`<h3>`
- Семантические теги: `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
- Обязательный `<h1>` на каждой странице с ключевым запросом

### 1.3 Изображения
- Alt-тексты для всех 35+ изображений с ключевыми словами
- Конвертация PNG → WebP с `<picture>` fallback
- Сжатие: 25 МБ → ~5-8 МБ
- `srcset` для адаптивных размеров

### 1.4 Structured Data (расширение)
Текущее: LocalBusiness, FAQPage. Добавить:
- Product schema для каждого типа штор
- BreadcrumbList для навигации
- AggregateRating с привязкой к отзывам
- Offer schema с ценовыми диапазонами

### 1.5 Sitemap
- Все страницы, языковые версии, `<image:image>` для фото
- `lastmod`, `changefreq`, `priority`

## Секция 2: Конверсии и UX

### 2.1 Формы заявок
- Минимум полей: имя + телефон/WhatsApp
- Sticky CTA на мобильных (фиксированная кнопка внизу)
- Мультиканальность: форма + WhatsApp + звонок
- Микрокопирайтинг: «Бесплатный замер за 15 минут» вместо «Оставить заявку»

### 2.2 Элементы доверия
- Отзывы с фото и именами (карусель)
- Портфолио «До/После»
- Блок гарантий: «Бесплатный замер → Пошив 4-5 дней → Бесплатная установка»
- Контакты: адрес + карта + фото шоурума

### 2.3 Структура страницы (воронка)
1. Хиро-блок с УТП и CTA
2. Каталог/типы штор с ценами «от»
3. Процесс работы (3-4 шага)
4. Портфолио / примеры работ
5. Отзывы
6. FAQ
7. Контакты + форма

### 2.4 Мобильная оптимизация
- Click-to-call для телефона
- Click-to-WhatsApp кнопка
- Удобные формы на мобильных

## Секция 4: Техническая перестройка

### 4.1 Производительность
- PNG → WebP (экономия 60-70%), lazy-loading для всех изображений
- Удаление неиспользуемых JS-модулей (932 КБ → ~300-400 КБ)
- Замена jQuery 1.10.2 на ванильный JS
- Вынос 80 КБ inline-стилей в внешние CSS
- Ограничение шрифтовых начертаний (29 файлов → необходимый минимум)

### 4.2 Accessibility
- Alt-тексты, ARIA-метки
- Семантические заголовки
- Фокус-индикаторы
- Цветовой контраст

### 4.3 Инфраструктура
- Генератор: Hugo или 11ty (шаблоны вместо копирования HTML)
- Скрипт сборки: оптимизация изображений + минификация
- Деплой: GitHub Pages / Netlify / Cloudflare Pages (CDN + SSL)

### 4.4 Мониторинг
- Google Search Console
- Web Vitals (LCP, FID, CLS)
- Цели в Yandex.Metrika и GTM

## Приоритеты реализации

1. **Фаза 1 (перед запуском):** SEO-фундамент (мета-теги, alt-тексты, семантика, structured data, sitemap)
2. **Фаза 2 (перед запуском):** Конверсионные правки (формы, CTA, элементы доверия)
3. **Фаза 3 (перед запуском):** Оптимизация изображений (WebP, сжатие)
4. **Фаза 4 (после запуска):** Посадочные страницы и контентное расширение
5. **Фаза 5 (после запуска):** Техническая перестройка (генератор, удаление jQuery, вынос стилей)
